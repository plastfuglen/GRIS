# 🐷 GRIS

Et nettleserbasert kortspill inspirert av det klassiske selskapsbordet «Gris» (også kjent som «Pig» på engelsk).

Spill det her: **[plastfuglen.github.io/gris](https://plastfuglen.github.io/GRIS)**

---

## Regler

Alle spillere får et antall kort på hånden. Hver runde velger du ett kort og sender det videre til neste spiller — alle sender samtidig.

Målet er å samle **alle kortene dine av samme sort** (f.eks. bare ♥). Når du har det, legger du tommelen diskret ned på bordet. De andre må legge ned tommelen så raskt som mulig.

Den **siste** til å legge ned tommelen får en bokstav:  
**G** → **R** → **I** → **S**

Staver du **GRIS** er du ute!

---

## Funksjoner

- 🤖 Tre NPC-motstandere med realistiske reaksjonstider
- ⏱️ Reaksjonstider vises etter hver runde — for deg og NPC-ene
- 🎮 Fire vanskelighetsgrader: Lett, Medium, Vanskelig, Ekstrem
- 🃏 Valgbar håndstørrelse: 4–8 kort
- ⌨️ Støtter klikk og mellomrom for å legge ned tommelen
- 📱 Fungerer på mobil og desktop

---

## Teknisk

Én enkelt HTML-fil — ingen avhengigheter, ingen byggsteg, ingen server.

---

## Kjør lokalt

```bash
git clone https://github.com/plastfuglen/gris.git
cd gris

```
